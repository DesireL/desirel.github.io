Linux命令（一）

linux 开发

作为一个猿类，熟练使用linux命令真的太重要啦，尤其是最近在折腾NAS，需要使用各种命令进行相关操作。经常发生的情况是用过某个命令，完了给忘了，用到了又要搜索😂，而且搜到的又不一定是合适的，所以有必要记录一波常用的linux相关命令以及折腾的过程中遇到的一些坑。

登录linux

　　登录命令 ssh user@ip ，登录过程中可能会出现下面错误：Unable to negotiate with 192.168.1.1 port 22: no matching host key type found. Their offer: ssh-dss ，遇到这种情况是在折腾NAS的时候终端登录之后用 reboot 重启了NAS，结果重新登录出错无法登录，搜索到两个网页：

  • 方法一（未解决问题）:ssh -oHostKeyAlgorithms=+ssh-dss user@ip

这个解决办法并没有实际解决（不知道是不是系统影响）。

  • 方法二（解决问题）： ssh -o KexAlgorithms=diffie-hellman-group14-sha1 -oHostKeyAlgorithms=+ssh-dss user@ip
