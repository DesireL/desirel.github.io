博客

博客，仅音译，英文名为Blogger,为Web Log的混成词。它的正式名称为网络日记；又音译为部落格或部落阁等，是一种通常由个人管理、不定期张贴新的文章的网站。

　　博客网站真的是多如牛毛，程序员常用的博客有博客园和CSDN，但是对于我这种强迫症来说，很难去选择一个地方写博客，我能想到选个网站写博客之后过不了多久就会因为账号设置、网站设置之类的犯强迫症，浪费更多的时间在无用的折腾上面。（其实主要的原因还是因为懒得写😂😂 （逃））

　　有史以来写作之类的真的是让人头疼，打小开始写作文的时候就是靠抄作文书度过来的想想也是够拼，好不容易坚持到不念书了发现有时候还得写东西，真的是让人头疼啊（这是病，得治！），所以药不能停。

　　既然有了写作的这个需求，舍友聂逗比前段时间搭了个博客之后又双叒叕来安利我，so前几天抽了一点时间自己折腾搭博客开始写（zhuang）文（bi）。

MarkDown

认识与入门MarkDown：Markdown是一种可以使用普通文本编辑器编写的标记语言，通过简单的标记语法，它可以使普通文本内容具有一定的格式。

　　去年大概是十一月份的时候，工作任务少有比较多的闲余时间，随便逛网站的时候无意中发现 MarkDown这个东西，当时就被它神奇的地方吸引了（过太久没用这货，我也忘了当初是哪一点吸引了我）写了几个小内容练了下手，现在开始写博客又要重新捡起来了。
